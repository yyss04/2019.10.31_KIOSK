# [Team Project] 키오스크 무인주문기

> 음식메뉴를 주문하고, 관리자모드로 메뉴를 관리할 수 있는 무인주문사이트 입니다.

## 프로젝트소개
  * 프로젝트 기간 : 2019-10-31 ~ 2019-12-04

## 개발환경
  * java 1.8
  * spring 3.x
  * oracle DB 11g
  * tomcat 8.5
  * jquery 3.2.1
  * html5, css3, js, bootstrap

## 주요기능
  * 메뉴 주문, 수량조절, 취소기능
  * 메뉴 장바구니기능
  * 관리자 로그인, ID기억
  * 관리자 모드에서 메뉴 CRUD 
  * 관리자 모드로 추천메뉴선택
